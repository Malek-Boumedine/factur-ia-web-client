Affiche les plans de la plateforme (GET /abonnements/) en cartes : nom, tarif mensuel et limites (utilisateurs, factures/mois). Le plan actuellement souscrit par l'entreprise active (GET /abonnements/me, statut actif) est mis en évidence. Sans souscription active, un bandeau signale la dernière souscription connue et son statut (expiré, suspendu, annulé). Changement de plan : chaque plan différent du plan actuel porte un bouton « Choisir ce plan » (POST avec confirmation vers `abonnement_changer`), visible uniquement pour les administrateurs de l'entreprise (`is_entreprise_admin`), l'API restant juge de vérité. Échéance et prolongation : la carte du plan actuel affiche l'échéance (`date_fin`, « Sans échéance » pour le gratuit qui n'expire pas) et, pour un admin d'entreprise sur un plan payant (`date_fin` présente — le critère même du 409 de l'API), un bouton « Prolonger d'un mois » (POST avec confirmation vers `abonnement_prolonger`). L'expiration est paresseuse côté API : une entreprise échue apparaît déjà rebasculée sur le gratuit dans /abonnements/me. {% endcomment %} {% block title %}Abonnements — Factur-IA{% endblock %} {% block content %}
